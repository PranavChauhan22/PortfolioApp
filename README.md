# PortFolio App
Short resume app that acts as my pocket portfolio.

## Direct Access through Expo Go
[Expo Go](https://exp.host/@pranavchauhan/portfolio)
## Installation

Use the package manager [npm](https://docs.npmjs.com/) to install foobar.

```bash
npm i -g expo-cli
```

## Usage

```Javascript
npm start
```
